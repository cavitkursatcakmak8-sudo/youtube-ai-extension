# YouTube AI Extension

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/cavitkursatcakmak8-sudo/youtube-ai-extension.git
   cd youtube-ai-extension
   ```
2. Install the required packages:
   ```bash
   npm install
   ```

## Usage

1. Start the application:
   ```bash
   npm start
   ```
2. Open your browser and navigate to `http://localhost:3000` to use the extension.
